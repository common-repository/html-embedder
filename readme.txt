== Description ==

This plugin allows you to easily embedd html markup into your posts. You can embed things such as

* YouTube Videos
* Google Map maps
* Arbitrary html

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
1. Activate the plugin in the admin panel
1. Add custom fields to your post. The field will contain the html you with to embed.
1. Embed into your posts using the contruct [field name="NameOfCustomField"]
